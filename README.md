### 🌟 Featured Project — Emergency Rescue Node Communication System

> An offline disaster communication platform that works when cellular and internet networks fail. Built as my Final Year Project to address the 2025 Pakistan floods that destroyed 4,300+ telecom towers.

**Achievements:**
- 🎯 **97.3%** packet delivery rate across multi-hop mesh
- ⚡ **5.4 sec** end-to-end SOS delivery latency
- 🔋 **11.2 hours** continuous battery operation
- 📡 **3-node** LoRa mesh network at 433 MHz

**Tech Stack:** LoRa ESP32 V2 • SX1276 • Embedded C/C++ • HTML/CSS/JS • Supabase PostgreSQL • Browser Geolocation API • RadioLib • WebSockets
